# USspending-contracts-analysis-dashboard
A Power BI dashboard analyzing federal contracting trends, obligations, set-asides, and small business participation.
This project analyzes federal contract awards to identify trends, small business participation, NAICS categories aligned with American Development Partners’ capabilities, and high-value opportunities.

Key Questions Answered

How much federal contracting value is going to small and minority owned businesses?

Which agencies have the highest spending in ADP-related NAICS codes?

Which opportunities represent the strongest strategic fit?

Where are contracts geographically concentrated?
